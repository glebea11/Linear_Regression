
## Линейная регрессия — базовый алгоритм машинного обучения для задач обучения с учителем, который используется для предсказания числовых значений на основе входных данных

Математический вид линейной регрессии

### Линейную регрессию можно описать через формулу y = a + bx,  
где  

- y — зависимая переменная  
- x — независимая переменная  
- a — свободный член (смещение по оси Y, или значение y, когда x равно 0)  
- b — угловой коэффициент (наклон), показывающий, на сколько в среднем увеличивается y при увеличении x на единицу
